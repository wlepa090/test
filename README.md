# Было создано GUI-приложение «GitHub User Finder» с использованием GitHub API, сохранением избранных пользователей и Git
#Как использовать API: API (Application Programming Interface) — это «программный интерфейс приложения». Простыми словами, это договор или меню для общения между двумя компьютерными программами.
Представь, что ты сидишь в ресторане. Ты — это клиент (твоя программа), а кухня — это сервер (например, сервер GitHub). Ты не можешь просто зайти на кухню и начать готовить. Тебе нужно сделать заказ.
#Примеры использования API: Поиск по логину, поиск по ключевому слову, поиск с уточнением типа, получение полной информации о пользователе
